JQuery JavaScript Library 1/2

JQuery is a bunch of JavaScript functions piled up together to make things easier for web designers.

JQuery is a shorthand to the longer JavaScript object methods in JavaScript. It uses CSS selectors to get elements

Example :

$("#test").html() === document.getElementById("test").innerHTML

$("input[name='password']").val() === document.querySelector("input[name='password']").value

JQuery JavaScript Library 2/2

It also has a bunch of useful functions that do not readily exist in JavaScript like:

$("#test").animate()

$("#test(gt:2)").css('display', 'none')

$.post()

$("#test").fadeIn()

... etc ...
