What is PHP 1/2

PHP (recursive acronym for PHP: Hypertext Preprocessor) is a widely-used open source general-purpose scripting language that is especially suited for web development and can be embedded into HTML. (http://www.php.net/manual/en/intro-whatis.php)

Things to note:
  • Scripting Language: pros and cons of scripting languages, but it can also be Object Oriented.
  • Embedded into HTML: can be inserted into HTML
  • Preprocessor?... like "#define" in C or "private static final" in Java. But can do more then just preprocess.
Pros:
  • Apache and PHP are attached at the hip.
  • I have seen a version for PHP on every OS that I can think of.
